## 🛠️ Troubleshooting Common Issues

**❌ Gui Does Not Appear After Injection**
The **Meepcity Script Gui** occasionally fails to render when the executor loads before the game client finishes its handshake.
**Solution:** Wait until you are fully inside the lobby, then re-execute the script. Ensure your executor is updated to the latest build.

**❌ Auto Farm Stops Mid-Cycle**
The **Meepcity Script Auto Farm** module can stall if the game pushes a micro-update that shifts task coordinates.
**Solution:** Rejoin the server and re-inject. Check the changelog above for any hotfix notes addressing the current game version.

**❌ Copy Avatar Returns a Blank Outfit**
**Meepcity Script Copy Avatar** may return empty data if the target player is still loading assets.
**Solution:** Wait a few seconds after the target player fully appears, then click again. Avoid targeting players who just teleported in.

**❌ Mobile Controls Overlap Game UI**
The **Meepcity Script Mobile** layout can collide with native buttons on certain screen sizes.
**Solution:** Adjust `GuiScale` down to `0.8` in `config.lua` or drag the panel to a different screen corner.

**❌ Script Causes Lag Spikes**
Running too many modules simultaneously can strain lower-end hardware.
**Solution:** Disable non-essential features like the **Meepcity Script Troll** suite and `DebugMode`. Close background applications.
